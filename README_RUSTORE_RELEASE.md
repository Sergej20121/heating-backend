# Подготовка к релизу в RuStore

## Что уже усилено в этой версии
- production-конфиги вынесены в `.env.production.example`
- mock-оплата в production по умолчанию отключена
- rate limit вынесен в настройки окружения
- массовые уведомления и объявления отправляются батчами
- усилена проверка OTP и паролей
- добавлены индексы Prisma для горячих запросов

## Что обязательно сделать перед боевым релизом
1. Подключить реальный платёжный шлюз вместо mock-оплаты.
2. Выполнить миграцию Prisma после обновления схемы.
3. Заполнить production `.env` реальными доменами и секретами.
4. Собрать Android App Bundle и проверить release-сборку на реальном устройстве.
5. Прогнать smoke-тесты: регистрация, вход, показания, начисления, уведомления, заявки.

## Команды
```bash
npm install
npx prisma generate
npx prisma migrate dev
npm run build
npm run start:prod
```
